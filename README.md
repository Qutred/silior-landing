# Silior Store Landing Page

A stylish, responsive landing page for **Silior Store** — an online perfume shop focused on offering popular fragrances (Zara, Pink, Victoria’s Secret) at accessible prices in Ukraine.

The page is designed as a single-page marketing website with smooth scrolling, animated content blocks, featured products, customer reviews, FAQ, and direct call-to-action links to Instagram for ordering.

## ✨ Features

- **Modern hero section** with clear value proposition and CTA
- **Sticky header** that changes style on scroll
- **Product catalog cards** with pricing and discount display
- **Why choose us** benefits section
- **About brand** storytelling section
- **Customer reviews** block for social proof
- **FAQ accordion** with interactive expand/collapse behavior
- **Animated UI elements** powered by AOS (Animate On Scroll)
- **Responsive design** optimized for desktop and mobile screens

## 🛠 Tech Stack

- **HTML5** — semantic page structure
- **CSS3** — custom styling, layout, and responsive behavior
- **Vanilla JavaScript** — interactivity (header state, FAQ accordion, hover effects)
- **AOS Library** — scroll-triggered animations
- **Google Fonts** — Playfair Display + Montserrat

## 🚀 Run Locally

No build tools required.

1. Clone/download the project
2. Open `index.html` in your browser

Or use VS Code Live Server for auto-reload during development.

## 🎯 Business Goal

This landing page is built to convert visitors into buyers by:

- communicating trust and value,
- showcasing popular products with attractive pricing,
- reducing friction via direct ordering through Instagram.
